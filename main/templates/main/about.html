About Our Platform

A smart platform connecting clients with skilled writers for seamless collaboration.

🚀 Features

  • Role-based system (Client, Writer, Admin)
  • Portfolio showcase
  • Like & Comment system
  • Hire writers easily
  • Real-time chat

⚙️ How It Works

Writers publish content on the platform. Clients explore the portfolio, evaluate writers, and hire them. Once hired, both can communicate through chat.

👥 User Roles

  • Client: Explore & hire writers
  • Writer: Create & manage content
  • Admin: Manage platform

⚠️ Limitations

  • No payment integration
  • No file sharing in chat
  • Polling-based real-time system
